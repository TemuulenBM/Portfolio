# temuulen.dev

My personal portfolio site. Built with Next.js 16, React 19, and Three.js for the 3D background.

## Stack

- **Framework:** Next.js 16 (App Router)
- **Styling:** Tailwind CSS v4
- **3D:** React Three Fiber / Three.js
- **Fonts:** Space Grotesk + JetBrains Mono
- **Hosting:** Vercel

## Run locally

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
```
